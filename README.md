# Retail Data Warehouse (BigQuery)

## Overview
Designed an end-to-end data warehouse solution for a retail business using BigQuery.

## Architecture
- ELT pipeline
- Star schema design
- Cloud-based data warehouse

## Tools
- SQL
- BigQuery
- Excel / CSV datasets

## Features
- Sales analysis
- Profitability insights
- Customer segmentation

## Outcome
Enabled data-driven decision-making through structured data and BI-ready models.
